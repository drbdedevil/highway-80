Игра "Highway 80" про гонщика 80-х годов. У игрока есть 60 секунд, чтобы доехать до финиша. На пути гонщика появляется очень плотный трафик, который придётся объезжать. Старайтесь не выезжать за трассу, потому что это замедлит вас! Также на пути вам могут встретиться удивительные машины!

Управление: A - поворот влево, D - поворот вправо
